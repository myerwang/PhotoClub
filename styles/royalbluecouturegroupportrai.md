# Style: Royal Blue Couture Portrait

## Source Prompt

Create a royal-blue couture editorial portrait using exactly the selected subject count. Use deep sapphire and midnight-blue luxury styling, ornate baroque or studio interior cues, rich textile detail, polished campaign lighting, and refined couture atmosphere while preserving each selected face.

## Visual Rules

- Royal-blue couture campaign treatment: deep sapphire palette, ornate luxury interior or studio set, velvet or satin textures, crystal-like highlights, refined high-fashion styling, and polished editorial finish.
- Do not create a fixed group. Use exactly the selected subject count and never add extra models, duplicated faces, or unrelated companions.

## Composition

- For one selected subject, create a single-person couture portrait. For multiple selected subjects, arrange only those selected people in a cohesive editorial composition.
- Keep every selected face fully visible, uncropped, and identity-consistent.

## Lighting And Color

- Deep blue monochrome-leaning color design, soft diffused studio lighting, crisp textile detail, controlled highlights, polished skin texture, and moody luxury contrast.
